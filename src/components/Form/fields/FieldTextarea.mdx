# FieldTextarea

## Usage

Use this component for entering multiple lines of text when you want to restrict the user's entry to plain text. If the user is allowed to use bold, italics and other rich text markup, use the [FieldRichTextarea](../?path=/docs/forms-fieldrichtextarea--docs) component.

The `size` of the input area will signal to the user how much information they should enter into the field. Choose a size that is sufficient to display the expected input.
