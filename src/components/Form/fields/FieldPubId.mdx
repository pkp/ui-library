# FieldPubId

## Usage

This component is a special field for setting a pub id, like a URN, based on pre-configured settings in a pub id plugin. When the plugin settings permit a pub id to be customized directly, a [FieldText](../?path=/docs/forms-fieldtext--docs) should be used instead.
