# Dropdown Actions

## Usage

This component displays a dropdown menu with a list of actions. To show an ellipsis menu, set `displayAsEllipsis` to `true`. When the ellipsis menu is enabled, the `label` prop provides descriptive text for the "More Options" icon. If the ellipsis menu is not used, the `label` prop will be the text shown on the dropdown button.
