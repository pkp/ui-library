# Dropdown Actions

## Usage

This component renders a dropdown menu that can display a list of actions. If the `name` prop is supplied, it will be used as the button's label. To use an ellipsis menu, set `isEllipsis` to `true` and provide the `ariaLabel` for accessibility.
