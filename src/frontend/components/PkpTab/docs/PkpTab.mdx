# PkpTab

An accessible tab component for organizing content into switchable panels.

<Primary />
<Controls />

## HTML Structure

```html
<div class="PkpTabRoot">
	<!-- Tab buttons -->
	<div class="PkpTabList">
		<button class="PkpTabTrigger" data-state="active">Abstract</button>
		<button class="PkpTabTrigger" data-state="inactive">Details</button>
		<button class="PkpTabTrigger" data-state="inactive">References</button>
	</div>

	<!-- Tab content panels -->
	<div class="PkpTabContent" data-state="active">Abstract content here...</div>
	<div class="PkpTabContent" data-state="inactive">Details content here...</div>
	<div class="PkpTabContent" data-state="inactive">
		References content here...
	</div>
</div>
```
