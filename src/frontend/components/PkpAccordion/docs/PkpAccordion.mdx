# PkpAccordion

An accessible accordion component for expandable content sections.

<Primary />
<Controls />

## HTML Structure

```html
<div class="PkpAccordionRoot">
	<div class="PkpAccordionItem" data-state="open">
		<h3 class="PkpAccordionHeader">
			<button class="PkpAccordionHeader__trigger">
				<span class="PkpAccordionHeader__content">Section Title</span>
				<span class="PkpAccordionHeader__indicator">
					<svg viewBox="0 0 24 24">
						<polyline points="6 9 12 15 18 9" />
					</svg>
				</span>
			</button>
		</h3>
		<div class="PkpAccordionContent">Content goes here...</div>
	</div>
</div>
```

## PkpAccordionHeader Props

| Prop | Type     | Default | Description                              |
| ---- | -------- | ------- | ---------------------------------------- |
| `as` | `String` | `'h3'`  | The HTML element for the header wrapper. |

## Slots

### `#indicator`

The `#indicator` slot exposes an `{ open }` prop for state-aware content. The default slot content is a `ChevronDown` icon that rotates via CSS when the accordion is open.

```vue
<PkpAccordionHeader>
	Section Title
	<template #indicator="{ open }">
		{{ open ? 'Hide' : 'Show' }}
	</template>
</PkpAccordionHeader>
```

To hide the chevron entirely, provide an empty `#indicator` slot:

```vue
<PkpAccordionHeader>
	Section Title
	<template #indicator />
</PkpAccordionHeader>
```
