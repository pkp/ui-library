# PkpDialog

An accessible modal dialog for confirmations and user interactions.

<Primary />
<Controls />

## HTML Structure

```html
<!-- Dialog container (rendered in portal) -->
<div class="pkpDialog">
	<!-- Overlay/backdrop -->
	<div class="pkpDialog__overlay"></div>

	<!-- Centering container -->
	<div class="pkpDialog__position-container">
		<!-- Dialog content panel -->
		<div class="pkpDialog__content pkpDialog__content--default">
			<!-- Close button (optional) -->
			<button class="pkpDialog__close" aria-label="Close">
				<svg class="PkpIcon">...</svg>
			</button>

			<!-- Title -->
			<h2 class="pkpDialog__title">Submit Article</h2>

			<!-- Body/message -->
			<div class="pkpDialog__body">
				Are you sure you want to submit this article?
			</div>

			<!-- Action buttons -->
			<div class="pkpDialog__action-buttons">
				<button class="PkpButton">Confirm</button>
				<button class="PkpButton">Cancel</button>
			</div>
		</div>
	</div>
</div>
```
