# PkpTextarea

An accessible textarea component with label support.

<Primary />
<Controls />

## HTML Structure

```html
<div>
	<label class="PkpTextarea__label" for="unique-id">Your Comment</label>
	<textarea
		id="unique-id"
		class="PkpTextarea__element"
		placeholder="Enter your comment..."
	></textarea>
</div>

<!-- With visually hidden label (still accessible to screen readers) -->
<div>
	<label class="PkpTextarea__label sr-only" for="unique-id">Your Comment</label>
	<textarea id="unique-id" class="PkpTextarea__element"></textarea>
</div>
```
