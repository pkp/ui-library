# DashboardPage page

Used for listing submissions for editor and author as well as review assignment for reviewers.

<ArgTypes />

## Extension points for plugins

For more details and examples how to use them, check out [Plugins guide](..?path=/docs/guide-plugins--docs)

Name of the pinia store is `dashboard`.

- `getLeftControls`: Add additional controls above the table on left, next to `Filters` button.
- `getRightControls`: Add additional controls above the table on right, next to the Search field.
- `getColumns`: Add/Change which columns to render.
- `getEditorialActivityForEditorialDashboard`: Possibility to modify/add content in editorial activity column for Editorial Dashboard (for Editors).
- `getEditorialActivityForMySubmissions`: Possibility to modify/add content in editorial activity column for My Submissions dashboard (for Authors).
- `getEditorialActivityForMyReviewAssignments`: Possibility to modify/add content in editorial activity column for My Review Assignment dashboard (For Reviewers).
- `getReviewActivityIndicatorProps` - Provides configuration for individual Review Indicator (circle indicating status). This is unlikely use case for plugin, but its available.
- `getReviewActivityIndicatorPopoverProps` - Provides configuration for the popup for the popup, which gets displayed after clicking on the circle review indicator. This is unlikely use case for plugin, but its available.
