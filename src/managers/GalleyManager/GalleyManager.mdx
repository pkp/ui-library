# Galley manager

Listing of Galleys. Connected to legacy modals for individual actions. In future it will be replace with the Vue.js UIs.

<ArgTypes />

## Extension points for plugins

For more details and examples how to use them, check out [Plugins guide](..?path=/docs/guide-plugins--docs)

Name of the pinia store is `GalleyManager`.

- `getTopItems`: Add/Change items in top right, where the `Order` button is.
- `getBottomItems`: Add/Change items in top right, where the `Add galley` button is.
- `getItemActions`: Add/Change list of actions that are displayed after clicking 3 dots button.
- `getColumns`: Add/Change the columns being displayed in the table.
- `getManagerConfig`: GalleyManager configuration. Its unlikely use case to adjusting that from plugin, but its available.
